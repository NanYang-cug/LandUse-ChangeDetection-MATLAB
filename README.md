## 📖 项目简介
基于 MATLAB 的土地利用分类与变化检测系统，支持：
- 读取并预处理多光谱遥感影像  
- 基于 SVM/KNN 的监督分类  
- 后分类比较法的变化检测  
- 分类精度评估与面积统计
